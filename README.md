# Remote IDE



## First run

1. Install Docker version 24 or up.

2. Set password in `remote-ide/assets/server-config.yaml`

3. Build image:

   ```
   docker build --tag adrosar/remote-ide .
   ```



## Up

```
docker compose up -d
# OR
docker-compose up -d
```



## Down

```
docker compose down
# OR
docker-compose down
```
