# Remote IDE

## Run
1. Install Docker version 24
2. Run the script `./up.sh`
3. Open a web browser and go to [127.0.0.1:8080](http://127.0.0.1:8080)
4. The password is in file `/config/config.yaml`

## Shutting down
Execute the script `./down.sh`
